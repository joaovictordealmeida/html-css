Trechos de código

Exemplo de código em Python:

                    
num = int(input('Digite um número: '))
if num % 2 == 0:
    print(f'O número {num} é par')
else:
    print(f'O número {num} é ímpar')
print('Fim do programa!')

No código acima, usei 2 tags, uma delas foi a CODE, que é usada para delimitar um código. E a outra foi a PRE, que é usada para manter o texto pré-formatado (Ambas semânticas).

Tags unidas, jamais serão vencidas!

Quando uma tag está dentro de outra, elas juntam suas forças para tornar um novo resultado, por exemplo, neste texto: "Tags unidas, jamais serão vencidas", foi usada a tag EM (para transformar em itálico) e também foi usada a tag STRONG (para transformar em bold).

Citações

Tem dois tipos de citações que podemos usar...

Com aspas " ":

Para usar aspas em uma citação, como por exemplo neste texto: Rosas são vermelhas, pode se usar a tag Q (Semântica), do inglês significa "Quote", em português "Citar"

Com deslocamento:

As com deslocamento são usadas normalmente em frases mais longas, por exemplo:

O PHP É UMA LINGUAGEM ´PARA A CRIAÇÃO DE SCRIPTS PARA A WEB DO LADO SERVIDOR EMBUTIDOS EM HTML, CUJO CÓDIGO-FONT É ABERTO.
Usando a tag BLOCKQUOTE (Semântica). Podemos também colocar o link do texto original, usando o parâmetro CITE, assim como no parágrafo.

Abreviações

Nesta frase: "Eu estou aprendendo HTML e CSS para criar sites" usei a tag ABBR (Semântica), ela serve para indicar o significado de alguma abreviação. Sendo assim, quando o usuário passa o mouse em cima da sigla (no meu caso "HTML" e "CSS"), aparece o seu significado.

Texto invertido

Na frase a seguir: "TEXTO INVERTIDO" repare que o texto está invertido da direita para a esquerda, para isso, usei a tag BDO (em inglês "bi-directional override"), juntamente do parâmetro DIR que serve para indicar uma das possíveis direções, nesse caso LTR ("Left to Right") ou RTL("Right to Left")
